STEP 3: BUAT DATABASE
---------------------
1. Buka cPanel > MySQL Databases (atau phpMyAdmin)
2. Buat database baru: apaco318_MDN
3. Buat user database (atau gunakan existing)
4. Import database.sql ke database yang baru dibuat

   Via phpMyAdmin:
   - Pilih database apaco318_MDN
   - Tab "Import"
   - Choose file: database.sql
   - Klik "Go"

   Via Terminal/SSH:
   mysql -u username -p apaco318_MDN < database.sql

-------------------
Edit file config.php di root directory:

ðŸ”§ TROUBLESHOOTING:
================================================================================

PROBLEM: Cannot login
SOLUTION:
1. Check config.php - pastikan database credentials benar
2. Check database imported successfully
3. Clear browser cache
4. Try username: superadmin, password: admin123

PROBLEM: Upload image failed
SOLUTION:
1. Check folder uploads/ permission = 755
2. Check file size < 5MB
3. Check format: JPG, PNG, GIF, WEBP only
4. Check PHP upload_max_filesize setting

PROBLEM: Slider tidak muncul di landing page
SOLUTION:
1. Check slider status = Active
2. Check section yang dipilih
3. Slider TIDAK ditampilkan di section Layanan
4. Clear browser cache

PROBLEM: Admin menu tidak lengkap
SOLUTION:
- Beberapa menu hanya untuk Super Admin:
  * Hero Settings
  * Slider Management  
  * About Settings
  * Features
  * Contact Settings
  * Admin Management
  * Activity Logs
